CSS Урок 2: Классы и идентификаторы

Что такое class и id?

class — это имя для группы элементов, которым мы хотим задать одинаковый стиль.

id — это уникальное имя для одного конкретного элемента.

Пример в HTML:

<h1 class="title">Заголовок</h1>
<p class="text">Обычный текст</p>
<p class="text">Другой абзац</p>
<div id="important">Очень важный блок</div>

CSS для этих элементов:

.title {
  color: navy;
  text-align: center;
}

.text {
  color: #555;
  font-size: 18px;
}

#important {
  background-color: #ffffcc;
  padding: 10px;
  border: 2px solid #ff4444;
}
