Урок 1: Основы CSS

1. Что такое CSS?

CSS (Cascading Style Sheets) — каскадные таблицы стилей, язык, который управляет внешним видом веб-страниц.

2. Как подключать CSS к HTML?

Стили можно добавить тремя способами:

  • Inline — прямо в теге HTML (не рекомендуется):
    <p style="color: red;">Текст красного цвета</p>
  • Internal — внутри тега <style> в <head> (как здесь):
    <style> p { color: blue; } </style>
  • External — подключить отдельный CSS-файл:
    <link rel="stylesheet" href="styles.css">

3. Пример CSS-правил

Вот пример, который изменяет цвет и размер заголовка и текста:

h1 {
  color: blue;
  font-size: 32px;
  text-align: center;
}

p {
  color: gray;
  font-size: 18px;
}

4. Inline, Internal и External стили — кратко

Inline стили применяются к отдельному элементу и имеют самый высокий приоритет.

Internal стили применяются ко всей странице, задаются в <style> в <head>.

External стили — самый удобный и масштабируемый способ — это отдельный файл с расширением .css, который подключается в HTML.

5. Простое задание

Попробуй изменить стиль заголовка и параграфа в этом уроке, например, поменяй цвета и размер шрифта. Сделай так, чтобы твой сайт выглядел красиво и удобно для чтения.

/* Пример изменения стиля */
h1 { color: darkgreen; }
p { font-size: 20px; color: #555555; }

Если хочешь, могу помочь с проверкой твоих изменений — просто пришли код!
